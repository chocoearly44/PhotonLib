## Installation
### Maven
```xml
<dependency>
    <groupId>tk.thesuperlab</groupId>
    <artifactId>photonlib</artifactId>
    <version>VERSION</version>
</dependency>
```

### Gradle
```groovy
dependencies {
    implementation 'tk.thesuperlab:photonlib:VERSION'
}
```
